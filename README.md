# Cue King Snooker

A premium snooker and pool club booking platform blending physical gaming with digital precision. This application offers live table status, interactive time block reservations, a digital leaderboard, and a pro shop for snooker equipment.

## Technical Setup

This is a [Next.js](https://nextjs.org) project bootstrapped with `create-next-app`.

### Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

This project uses modern glassmorphism design principles, Tailwind CSS, and Framer Motion for smooth, interactive experiences.
